# **App Name**: Dynamic Product Ace

## Core Features:

- Product Media Gallery: Implement a product image carousel using default WooCommerce gallery support.
- Purchase Options: Implement two purchase modes as radio buttons: Single Drink Subscription and Double Drink Subscription, dynamically updating flavor selectors and pricing.
- Flavor Selection: Implement dynamic flavor selection based on chosen plan
- Dynamic Pricing: Display dynamic pricing and sales
- "What’s Included" Box: List items that will be included with the plan dynamically based on chosen plan
- Add to Cart Logic: Add to cart logic to take flavor selections into account when placing the correct variant in the cart via AJAX.
- Upselling Opportunities: Implement cart customizations, recommending related products or optional extras that enhance the value of the order.

## Style Guidelines:

- Primary color: Soft beige (#F5F5DC) to create a welcoming and earthy aesthetic, aligning with the natural elements often associated with beverage products.
- Background color: Off-white (#FAFAFA), nearly indistinguishable from pure white to give a refreshing, airy feel and ensure high readability.
- Accent color: Muted gold (#D4A27A), complementing the beige and adding a touch of sophistication that reflects the high-quality nature of the drinks.
- Body text: 'Inter', a sans-serif typeface to give the page a contemporary feel, ensures high readability.
- Headlines: 'Belleza', a humanist sans-serif for a personal and artisanal touch; but use 'Alegreya' for longer body text if needed.
- Minimalist icons representing the subscription benefits and drink flavors. Line icons for clarity.
- Ensure responsiveness and maintain a clear visual hierarchy across different devices
